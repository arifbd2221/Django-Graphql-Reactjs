# Django-Graphql-Reactjs

This web application is divided into backend and frontend part.

In the backend part several api's are avilable. All the api are designed and developend by django-graphql framework.
User can create profile,login,create music tracks,upload music track(to cloudinary server),like others music tracks,
edit profil, music tracks etc. All the api request are secured by JWT.

In the frontend part reactjs takes care of everything. Apollo client has been used to consume, request api's from the frontend.
Material design is used to give a good looking UI experience.
